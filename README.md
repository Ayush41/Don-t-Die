# Don't Die App

**Don't Die** is a fun and playful time-counter app that helps users track survival moments and milestones with a quirky twist. Whether you're counting the days of a challenge or celebrating small victories, this app makes tracking time enjoyable. 

Built using **React Native** and **Expo**, the app works seamlessly across **iOS**, **Android**, and **Web** platforms.

## Features

- **Track Survival Moments**: Keep track of how many days, hours, or minutes you've survived a challenge.
- **Customizable Timers**: Set up multiple timers for different events or milestones.
- **Achievements & Badges**: Unlock fun badges as you hit survival milestones.
- **Cross-Platform**: Compatible with iOS, Android, and Web.
- **Push Notifications**: Get fun reminders and survival notifications.
- **Interactive UI**: Engaging design to keep you motivated.
